# 공공기관 정원 뉴스 스크래퍼 (News Scraper)

## 프로젝트 개요
공공기관 정원 관련 정부동향, 뉴스, 법령개정 현황을 매일 자동 수집하여 이메일로 발송하는 프로그램

## 기술 스택
- **언어**: Python 3.11+
- **이메일**: Gmail SMTP
- **스케줄링**: GitHub Actions (매일 오전 8시 KST 자동 실행)
- **DB**: SQLite (중복 제거 및 이력 관리)
- **스크래핑**: requests, beautifulsoup4, feedparser
